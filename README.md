# Музыкальный стриминговый сервис

## Описание проекта
Веб-интерфейс для прослушивания музыки с возможностью создания собственных плейлистов. Сервис позволяет загружать треки, управлять очередью воспроизведения, ставить лайки и выбирать режим повтора. Все данные (плейлист, лайки, настройки) сохраняются в браузере с помощью `localStorage`.

## Функциональные возможности
- Встроенный аудиоплеер с обложками треков
- Отображение всех доступных треков с возможностью добавления в плейлист
- Управление очередью воспроизведения (добавление, удаление, переключение)
- Режимы повтора: без повтора, повтор одного трека, повтор всего плейлиста
- Лайки для треков (сохраняются в браузере)
- Кнопки навигации «Предыдущий» и «Следующий» с учётом режима повтора
- Автоматическое переключение на следующий трек по окончании текущего
- Сохранение состояния (плейлист, лайки, режим повтора, текущая позиция) в `localStorage`
- Адаптивный интерфейс (десктоп, планшет, мобильные устройства)

## Технологии
- HTML5
- CSS3 (Flexbox, Grid, медиа-запросы, тёмная тема)
- JavaScript (ES6, DOM-манипуляции, события)
- localStorage (сохранение состояния пользователя)
- Аудио API (HTML5 Audio)

## Запуск

2. Откройте файл `index.html` в любом современном браузере.
3. Для удобной разработки используйте Live Server в VS Code.
